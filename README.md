> ⚠️ WARNING! When you run this script, it can be used by everyone on the server! Be careful and use it on your test server or when there are no players on the server.

## Overview
The **nv_debug** script is designed for the **OX** or **Overextended** framework. To ensure functionality, the following scripts are required:
- **ox_lib**
- **ox_target**

## Features

### Entity Menu
- **Freeze/Unfreeze Position**
- **Set Entity Visibility (Alpha)**
- **Get Entity Info (Menu)**
  - Copy Entity Name
  - Copy Entity Position
  - Copy Entity Rotation
  - Copy Entity Heading
- **Vehicle Menu (Menu)**
  - Change Color
    - Black
    - Red
    - Orange
    - Blue
    - Green
    - Purple
    - Pink
  - Set Car Fuel
  - Fix Car
- **Ped Menu (Menu)**
  - Give Weapon (AK47 + Pistol)
  - Ragdoll Ped
  - Heal Ped
  - Clear All Tasks
  - Play Animation
  - Play Scenario
  - Kill Ped
- **Create OX Target**
- **Delete Entity**

### Player Menu
- Heal Yourself
- Kill Yourself
- Godmode On/Off
- Teleport
- Fast Travel
  - Casino
  - Maze Bank
  - Jewelry Store
  - Fleeca Bank
  - Maze Tower
  - Airport
  - Beach
  - Yacht
  - LSPD
  - Sub Urban

### Raycast Menu
- Copy Position
- Spawn Ped
- Spawn Vehicle
- Create Object
- Create BoxZone
- Create SphereZone
- Create Target (Box)
